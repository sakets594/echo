# Welcome to the ECHO Developer Diary

ECHO is a horror-puzzle game where you navigate a pitch-black environment using a Lidar scanner.

## How to Play
-   **Move**: WASD
-   **Scan**: Spacebar (Clap)
-   **Goal**: Find the Key and escape through the Exit. Avoid the Entity.
